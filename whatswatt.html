Whats Watt

A digital experience for understanding electricity consumption

What is it?
A physical installation transforming bicycling momentum into a digital representation of how much energy it takes to charge household electronics. An application is built for the users to check how much they would have to bike in order to use different household electronics in their home. The installation is made by using weight sensors attached to an arduino, which projects a live-feedback visualisation to a screen placed next to the bike, so the user can follow his progression. The application was build with a software called Titanium Appcelerator.

What I did?
I was mainly responsible for the communication between the application and the installation. Besides that I took part in the user research phase, I curated a Workshop with the target group, made a mock-up, build a prototype, coded the application, did user-testing and exhibited the installation at Sustainability Festival, Aarhus.
