BASF Urban Living

A campaign website for BASF

What is it?
BASF Urban Living is a website I coded for BASF during my traineeship at Virtual Identity. The website is set-up in tiles each representing a industry field of Urban Living that BASF is currently involved in.

What I did?
I developed the website in a framework called Biotope, which is a component based framework that allows for separating font and back-end. Technology-wise I mainly used html5, SASS and JQuery.
